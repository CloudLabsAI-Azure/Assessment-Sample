## Question
You need to create a new GitHub repository, which four actions would you perform in sequence to set up the new repo locally?<br>Move the appropriate answers and arrange them in the correct order. 

## Options
Option 1 : cd newrepo
Option 2 : git init
Option 3 : Install Git
Option 4 : mkdir newrepo
Option 5 : git repo –name newrepo
Option 6 : git create repo


## Answers
Option 3 : 3
Option 4 : 3
Option 1 : 3
Option 2 : 3
